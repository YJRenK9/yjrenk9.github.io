Title: Drawing Simulator


When it comes to a drawing machine, I decided to make one as it gave me an idea for project 3(it came to my mind). The more I think about the drawing machine, the more I think about possible brush modes to draw with. Hence, I came up with the name “Drawing Simulator” for project 3. My inspiration comes from using Tux Paint as a child. The Drawing Simulator allows users to use a variety of brush modes. As a software developer wannabe, creating this project gives me the incentive to create innovative brush modes and to gain some experience in developing “software applications”.
